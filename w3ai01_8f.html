SUBPROGRAM: W3AI01 UNPACK RECORD INTO IEEE F.P.

AUTHOR: JONES,R.E. ORG: W342 DATE: 89-10-17

ABSTRACT: UNPACKS A RECORD IN OFFICE NOTE 84 FORMAT AND CONVERT THE PACKED DATA TO IEEE REAL FLOATING POINT NUMBERS. THE OFFICE NOTE 84 DATA IS BIT FOR BIT THE SAME ON THE NAS-9050 AND THE CRAY.

PROGRAM HISTORY LOG: 89-10-20 R.E.JONES 90-02-02 R.E.JONES CHANGE TO CRAY FUNCTION FOR INTEGER*2, F.P. 90-10-11 R.E.JONES SPECIAL VERSION OF W3AI01 TO UNPACK RECORDS PACKED BY BIG VERSION OF W3AI00. WILL DO OLD AND NEW VERSION. 91-03-19 R.E.JONES MAKE SPECIAL VERSION OF W3AI01 TO UNPACK BIG RECORDS THE OPERATIONAL VERSION. 93-06-10 R.E.JONES INCREACE ARRAY SIZE TO 262144 WORDS. 98-03-10 B. VUONG REMOVE THE CDIR$ INTEGER=64 DIRECTIVE 98-11-17 Gilbert Changed to unpack into IEEE reals for the IBM SP

USAGE: CALL W3AI01 (PACK, REAL8, LABEL)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

PACK ARG LIST INTEGER ARRAY WITH DATA IN OFFICE NOTE 84 FORMAT TO BE UNPACKED.

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

REAL8 ARG LIST REAL ARRAY OF N WORDS. WHERE N IS GIVEN IN WORD 6 OF PACK. WORD 6 OF PACK MUST CONTAIN CENTER AND SCALING VALUES. LABEL ARG LIST SIX WORD INTEGER LABEL COPIED FROM PACK, 12 OFFICE NOTE 84 32 BIT ID'S THAT ARE STORED INTO six 64-bit words.

SUBPROGRAMS CALLED: NAMES LIBRARY

Q9IE32 W3LIB

REMARKS: LABEL AND PACK MAY BE EQUIVALENCED.

ATTRIBUTES: LANGUAGE: IBM XL FORTRAN MACHINE: IBM SP
