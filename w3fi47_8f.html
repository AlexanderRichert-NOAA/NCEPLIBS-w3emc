SUBPROGRAM: W3FI47 CONVERT LABEL TO OFF.

NO. 85 FORMAT (CRAY) AUTHOR: JONES, R.E. ORG: W342 DATE: 85-07-31

ABSTRACT: CONVERTS A OFFICE NOTE 85 LABEL IN IBM370 FORMAT TO OFFICE NOTE 85 CRAY FORMAT. ALL EBCDIC CHARACTERS ARE CONVERTED TO ASCII. CONVERTS BINARY OR CODED LABEL.

PROGRAM HISTORY LOG: 85-07-31 R.E.JONES 89-10-24 R.E.JONES CONVERT TO CRAY CFT77 FORTRAN 02-10-15 VUONG REPLACED FUNCTION ICHAR WITH MOVA2I

USAGE: CALL W3FI47(ILABEL,NLABEL)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

ILABEL ARG LIST 4 WORDS (32 BYTES) CHARACTERS ARE IN EBCDIC OR BINARY.

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

NLABEL ARG LIST 4 WORDS (32 BYTES), CHARACTERS ARE IN ASCII OR BINARY.

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY Y-MP8/832
