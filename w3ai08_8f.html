SUBPROGRAM: W3AI08 UNPK GRIB FIELD TO GRIB GRID PRGMMR: BOSTELMAN ORG: NMC421 DATE:90-07-31.

ABSTRACT: UNPACK A GRIB FIELD TO THE EXACT GRID SPECIFIED IN THE MESSAGE, ISOLATE THE BIT MAP AND MAKE THE VALUES OF THE PRODUCT DESCRIPTION SEC (PDS) AND THE GRID DESCRIPTION SEC (GDS) AVAILABLE IN RETURN ARRAYS.

PROGRAM HISTORY LOG: 88-01-20 CAVANAUGH 90-05-11 CAVANAUGH TO ASSURE THAT ALL U.S. GRIDS IN THE GRIB DECODER COMPLY WITH SIZE CHANGES IN THE DECEMBER 1989 REVISIONS. 90-05-24 CAVANAUGH CORRECTS SEARCHING AN IMPROPER LOCATION FOR GRIB VERSION NUMBER IN GRIB MESSAGES. 90-07-15 BOSTELMAN MODIIFED SUB. AI084 SO THAT IT WILL TEST THE GRIB BDS BYTE SIZE TO DETERMINE WHAT ECMWF GRID ARRAY SIZE IS TO BE SPECIFIED. 90-09-14 R.E.JONES CHANGE'S FOR ANSI FORTRAN, AND PDS VERSION 1 90-09-23 R.E.JONES CHANGE'S FOR CRAY CFT77 FORTRAN 90-12-05 R.E.JONES CHANGE'S FOR GRIB NOV. 21,1990 02-10-15 VUONG REPLACED FUNCTION ICHAR WITH MOVA2I

USAGE: CALL W3AI08(MSGA,KPDS,KGDS,KBMS,DATA,KPTR,KRET) INPUT ARGUMENT LIST: MSGA - GRIB FIELD - "GRIB" THRU "7777" CHAR*1
