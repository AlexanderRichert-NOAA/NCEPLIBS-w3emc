SUBPROGRAM: W3FI76 CONVERT TO IBM370 FLOATING POINT PRGMMR: REJONES ORG: NMC421 DATE:92-11-16.

ABSTRACT: CONVERTS FLOATING POINT NUMBER FROM MACHINE REPRESENTATION TO GRIB REPRESENTATION (IBM370 32 BIT F.P.).

PROGRAM HISTORY LOG: 85-09-15 JOHN HENNESSY ECMWF 92-09-23 JONES R. E. CHANGE NAME, ADD DOC BLOCK 93-10-27 JONES,R. E. CHANGE TO AGREE WITH HENNESSY CHANGES 95-10-31 IREDELL REMOVED SAVES AND PRINTS 98-03-10 B. VUONG REMOVE THE CDIR$ INTEGER=64 DIRECTIVE

USAGE: CALL W3FI76 (FVAL, KEXP, KMANT, NBITS) INPUT ARGUMENT LIST: PVAL - FLOATING POINT NUMBER TO BE CONVERTED KBITS - NUMBER OF BITS IN COMPUTER WORD (32 OR 64)

OUTPUT ARGUMENT LIST: KEXP - 8 BIT SIGNED EXPONENT KMANT - 24 BIT MANTISSA (FRACTION)

REMARKS: SUBPROGRAM CAN BE CALLED FROM A MULTIPROCESSING ENVIRONMENT.

ATTRIBUTES: LANGUAGE: IBM370 VS FORTRAN 77, CRAY CFT77 FORTRAN MACHINE: HDS 9000, CRAY Y-MP8/864< CRAY Y-MP EL2/256
