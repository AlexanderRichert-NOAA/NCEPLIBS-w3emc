SUBPROGRAM: W3FI63 UNPK GRIB FIELD TO GRIB GRID PRGMMR: FARLEY ORG: NMC421 DATE:94-11-22.

ABSTRACT: UNPACK A GRIB (EDITION 1) FIELD TO THE EXACT GRID SPECIFIED IN THE GRIB MESSAGE, ISOLATE THE BIT MAP, AND MAKE THE VALUES OF THE PRODUCT DESCRIPTON SECTION (PDS) AND THE GRID DESCRIPTION SECTION (GDS) AVAILABLE IN RETURN ARRAYS.

WHEN DECODING IS COMPLETED, DATA AT EACH GRID POINT HAS BEEN RETURNED IN THE UNITS SPECIFIED IN THE GRIB MANUAL.
