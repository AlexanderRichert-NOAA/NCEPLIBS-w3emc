SUBPROGRAM: W3FP13 CONVERT GRIB PDS EDITION 1 TO O.N.

84 ID PRGMMR: MCCLEES ORG: NMC421 DATE:91-10-07

ABSTRACT: CONVERTS GRIB VERSION 1 FORMATTED PRODUCT DEFINITION SECTION TO AN OFFICE NOTE 84 ID LABEL. FORMATS ALL THAT IS APPLI- CABLE IN THE FIRST 8 WORDS OF O.N. 84. (CAUTION ****SEE REMARKS)

PROGRAM HISTORY LOG: 91-10-07 ORIGINAL AUTHOR MCCLEES, A. J. 92-01-06 R.E.JONES CONVERT TO SiliconGraphics 3.3 FORTRAN 77 93-03-29 R.E.JONES ADD SAVE STATEMENT 94-04-17 R.E.JONES COMPLETE REWRITE TO USE SBYTE, MAKE CODE PORTABLE, UPGRADE TO ON388, MAR 24,1994 94-05-05 R.E.JONES CORRECTION IN TWO TABLES 96-08-02 R.E.JONES ERROR USING T MARKER 96-09-03 R.E.JONES ADD MERCATOR GRIDS 8 AND 53 TO TABLES 99-02-15 B. FACEY REPLACE W3FS04 WITH W3MOVDAT. 02-10-15 VUONG REPLACED FUNCTION ICHAR WITH MOVA2I

USAGE: CALL W3FP13 (GRIB, PDS, ID8, IERR ) INPUT ARGUMENT LIST: GRIB - GRIB SECTION 0 READ AS CHARACTER*8 PDS - GRIB PDS SECTION 1 READ AS CHARACTER*1 PDS(*)

OUTPUT ARGUMENT LIST: (INCLUDING WORK ARRAYS) ID8 - 12 INTEGER*4 FORMATTED O.N. 84 ID. 6 INTEGER 64 BIT WORDS ON CRAY IERR 0 - COMPLETED SATISFACTORILY 1 - GRIB BLOCK 0 NOT CORRECT 2 - LENGTH OF PDS NOT CORRECT 3 - COULD NOT MATCH TYPE INDICATOR 4 - GRID TYPE NOT IN TABLES 5 - COULD NOT MATCH TYPE LEVEL 6 - COULD NOT INTERPRET ORIGINATOR OF CODE SUBPROGRAMS CALLED: SPECIAL: INDEX, MOVA2I, CHAR, IOR, IAND, ISHFT

LIBRARY: W3LIB: W3MOVDAT, W3FI69, W3FI01

REMARKS: SOME OF THE ID'S WILL NOT BE EXACT TO THE O.N. 84 FOR LOCATING FIELD ON THE DATASET. THESE DIFFERENCES ARE MAINLY DUE TO TRUNCATION ERRORS WITH LAYERS. FOR EXAMPLE: .18019 SIG .47191 SIG R H FOR 36.O HRS WILL CONVERT TO: .18000 SIG .47000 SIG R H FOR 36.0 HRS !!!!!!!THE ABOVE ID'S NOW FORCED TO BE EXACT!!!!!!!!! IF J THE WORD COUNT IS GREATER THEN 32743, J IS STORED IN THE 12TH ID WORD. BITS 16-31 OF THE 8TH ID WORD ARE SET TO ZERO.

ATTRIBUTES: LANGUAGE: FORTRAN 90
