SUBPROGRAM: W3AI19 LINE BLOCKER SUBROUTINE PRGMMR: BOB HOLLERN ORG: NCO/NP12 DATE: 97-04-15.

ABSTRACT: FILLS A RECORD BLOCK WITH LOGICAL RECORDS OR LINES OF INFORMATION.

PROGRAM HISTORY LOG: 74-02-01 BOB ALLARD, AUTHOR 90-09-15 R.E.JONES CONVERT FROM IBM370 ASSEMBLER TO MICROSOFT FORTRAN 5.0 90-10-07 R.E.JONES CONVERT TO SUN FORTRAN 1.3
91-07-20 R.E.JONES CONVERT TO SiliconGraphics 3.3 FORTRAN 77
93-03-29 R.E.JONES ADD SAVE STATEMENT 94-04-22 R.E.JONES ADD XMOVEX AND XSTORE TO MOVE AND STORE CHARACTER DATA FASTER ON THE CRAY 97-04-15 Bob Hollern CORRECTED THE PROBLEM OF INIIALIZING NBLK TO @'S INSTEAD OF BLANKS

USAGE: CALL W3AI19 (LINE, L, NBLK, N, NEXT) INPUT ARGUMENT LIST: LINE - ARRAY ADDRESS OF LOGICAL RECORD TO BE BLOCKED L - NUMBER OF CHARACTERS IN LINE TO BE BLOCKED N - MAXIMUM CHARACTER SIZE OF NBLK NEXT - FLAG, INITIALIZED TO 0

OUTPUT ARGUMENT LIST: NBLK - BLOCK FILLED WITH LOGICAL RECORDS NEXT - CHARACTER COUNT, ERROR INDICATOR

EXIT STATES: NEXT = -1 LINE WILL NOT FIT INTO REMAINDER OF BLOCK; OTHERWISE, NEXT IS SET TO (NEXT + L) NEXT = -2 N IS ZERO OR LESS NEXT = -3 L IS ZERO OR LESS

EXTERNAL REFERENCES: XMOVEX XSTORE

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916-128, CRAY Y-MP8/864, CRAY Y-MP EL2/256
