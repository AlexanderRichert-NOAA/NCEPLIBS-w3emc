SUBROUTINE: W3FT17 CONVERT (95,91) GRID TO (3447) GRID AUTHOR: JONES,R.E.

ORG: W342 DATE: 94-05-03

ABSTRACT: CONVERT A SOUTHERN HEMISPHERE 1.0 DEGREE LAT.,LON. 95 BY 91 GRID TO A WAFS 1.25 DEGREE THINNED 3447 POINT GRID.

PROGRAM HISTORY LOG: 94-05-03 R.E.JONES

USAGE: CALL W3FT17(ALOLA,BTHIN,INTERP)

INPUT ARGUMENTS: ALOLA - 95 * 91 GRID 1.0 DEG. LAT,LON GRID SOUTHERN HEMISPHERE 8645 POINT GRID. INTERP - 1 LINEAR INTERPOLATION , NE.1 BIQUADRATIC

INPUT FILES: NONE

OUTPUT ARGUMENTS: BTHIN - 3447 POINT THINNED GRID OF S. HEMISPERE 3447 GRID IS FOR GRIB GRIDS 41-44.

OUTPUT FILES: ERROR MESSAGE TO FORTRAN OUTPUT FILE

WARNINGS:

  1. W1 AND W2 ARE USED TO STORE SETS OF CONSTANTS WHICH ARE REUSABLE FOR REPEATED CALLS TO THE SUBROUTINE. 10 OTHER ARRAYS ARE SAVED AND REUSED ON THE NEXT CALL.

RETURN CONDITIONS: NORMAL SUBROUTINE EXIT

SUBPROGRAMS CALLED: UNIQUE : NONE

LIBRARY:

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916-128, cRAY Y-MP8/864, CRAY Y-MP EL2/256
