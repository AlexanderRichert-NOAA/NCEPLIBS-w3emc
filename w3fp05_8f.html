SUBPROGRAM: W3FP05 PRINTER CONTOUR SUBROUTINE AUTHOR: R.E.JONES ORG: W342 DATE: 85-07-31.

ABSTRACT: PRINTS A TWO-DIMENSIONAL GRID OF ANY SHAPE, WITH CONTOURING, IF DESIRED. GRID VALUES ARE SCALED ACCORDING TO TO CONSTANTS SPECIFIED BY THE PROGRAMER, ROUNDED, AND PRINTED AS 4,3, OR 2 DIGIT INTEGERS WITH SIGN, THE SIGN MARKING THE GRID POSITION OF THE PRINTED NUMBER. IF CONTOURING IS REQUESTED, BESSEL'S INTERPOLATION FORMULA IS USED TO OPTAIN THE CONTOUR LINES. CONTOURS ARE INDICATED BY ALPHABETIC CHARACTERS RANGING FROM A TO H OR NUMERIC CHARACTERS FROM 0 TO 9. CONTOUR ORIGIN AND INTERVAL ARE SPECIFIED BY THE PROGRAMMER IN TERMS OF PRINTED VALUES.

PROGRAM HISTORY LOG: 89-10-13 R.E.JONES 92-05-02 R.E.JONES ADD SAVE

USAGE: CALL W3FP05 (RDATA,KTBL,CNST,TITLE,KRECT,KCONTR,LINEV,IWIDTH)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

RDATA ARG LIST REAL ARRAY OF GRID DATA TO BE PRINTED. KTBL ARG LIST INTEGER ARRAY WITH SHAPE OF ARRAY. CNST ARG LIST REAL ARRAY OF FOUR ELEMENTS, USED IN SCALING FOR PRINTING AND CONTOURING. TITLE ARG LIST IS A ARRAY OF 132 CHARACTERS OR LESS OF HOLLERITH DATA, 1ST CHAR. MUST BE BLANK. PRINTED AT BOTTOM OF THE MAP. KRECT ARG LIST 1 IF GRID IS RECTANGULAR, 0 OTHERWISE. KCONTR ARG LIST 1 FOR CONTOURING , 0 OTHERWISE. LINEV ARG LIST 0 IS FOR 6 LINES PER VERTICAL INCH, NON-ZERO 8 LINES PER VERTICAL INCH. IWIDTH ARG LIST NUMBER OF CHARACTERS IN PRINT LINE, 132 IS STANDARD PRINTER.

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

TAPE6 MAPS ON STANDARD FORTRAN PRINT FILE

SUBPROGRAMS CALLED: NAMES LIBRARY

ABS IABS MIN0 MOD SYSTEM

REMARKS: NORMAL SUBROUTINE RETURN, UNLESS NUMBER OF ROWS IS GREATER THAN 200, PRINTS ERROR MESSAGE AND EXITS.

ATTRIBUTES: LANGUAGE: CRAY CTF77 FORTRAN MACHINE: CRAY Y-MP/832
