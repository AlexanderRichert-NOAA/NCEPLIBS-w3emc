SUBPROGRAM: W3FA06 CALCULATION OF THE LIFTED INDEX AUTHOR: HOWCROFT,J.

ORG: W/NMC342 DATE: 78-07-01

ABSTRACT: GIVEN THE PRESSURE,TEMPERATURE AND RELATIVE HUMIDITY OF AN AIR PARCEL AT SOME POINT IN THE ATMOSPHERE, CALCULATE THE LIFTED INDEX OF THE PARCEL. LIFTED INDEX IS DEFINED AS THE TEMPERATURE DIFFERENCE BETWEEN THE OBSERVED 500MB TEMPERATURE AND THE SUPPOSED TEMPERATURE THAT THE PARCEL WOULD OBTAIN IF IT WERE LIFTED DRY-ADIABATICALLY TO SATURATION AND THEN MOVED MOIST ADIABATICALLY TO THE 500MB LEVEL.

PROGRAM HISTORY LOG: 78-07-01 J.HOWCROFT 89-01-24 R.E.JONES CHANGE TO MICROSOFT FORTRAN 4.10 90-06-08 R.E.JONES CHANGE TO SUN FORTRAN 1.3 91-03-29 R.E.JONES CONVERT TO SiliconGraphics FORTRAN 93-03-29 R.E.JONES ADD SAVE STATEMENT 95-09-25 R.E.JONES PUT IN W3 LIBRARY ON CRAY

USAGE: CALL W3FA06(P,T,RH,T5,TLI)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

P ARG LIST PARCEL PRESSURE IN MILLIBARS T ARG LIST PARCEL TEMPERATAURE IN DEGREES CELSIUS RH ARG LIST PARCEL RELATIVE HUMIDITY IN PERCENT T5 ARG LIST TEMPERATURE AT THE 500MB LEVEL IN DEG. CELSIUS

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

TLI ARG LIST LIFTED INDEX IN DEGREES CELSIUS TLI = 9.9999 ITERATION DIVERGES; RETURN TO USER PROGRAM

EXP ABS SIGN SYSLIB W3FA01 W3LIB
