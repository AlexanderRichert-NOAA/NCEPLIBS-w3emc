SUBPROGRAM: W3FP04 PRINT ARRAY OF DATA POINTS AT LAT/LON POINTS AUTHOR: HORODECK, J.

ORG: W324 DATE: 85-07-31

ABSTRACT: GIVEN AN ARRAY OF METEOROLOGICAL DATA AND CORRESPONDING LATITUDE/LONGITUDE POSITION FOR EACH DATA POINT, THESE DATA VALUES ARE PRINTED AT THEIR APPROXIMATE LATITUDE/LONGITUDE POSITIONS ON A POLAR STEREOGRAPHIC PROJECTION.

PROGRAM HISTORY LOG: 80-01-15 J. HORODECK 85-07-31 R.E.JONES CHANGE TO CDC FORTRAN 200 90-08-15 R.E.JONES CHANGE TO CRAY CFT77 FORTRAN

USAGE: CALL W3FP04(IFLD,ALAT,ALON,TITLE,IDIM,CMIL,CMIR,CMJB,CMJT, INUM,XFAC,IERR)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

IFLD ARG LIST REAL OR INTEGER FULLWORD ARRAY OF DATA POINTS ALAT ARG LIST REAL ARRAY OF LATITUDE POSITIONS (>0 FOR NH, <0 FOR SH) FOR THE DATA TO BE PLOTTED ALON ARG LIST REAL ARRAY OF LONGITUDES (WEST OF GREENWICH) TITLE ARG LIST INTEGER SIZE 10 ALPHANUMERIC ARRAY OF CHARACTERS FOR TITLE TO BE WRITTEN ON PRINTOUT IDIM ARG LIST INTEGER NUMBER OF DATA VALUES TO PLOT (SIZE OF ARRAYS IFLD, ALAT AND ALON) CMIL ARG LIST REAL LEFT SIDE OF GRID - MINIMUM COARSE MESH I COORDINATE (MINIMUM VALUE OF 1.0) CMIR ARG LIST REAL RIGHT SIDE OF GRID - MAXIMUM COARSE MESH I COORDINATE (MAXIMUM VALUE OF 65.0) CMJB ARG LIST REAL BOTTOM OF GRID - MINIMUM COARSE MESH J COORDINATE (MINIMUM VALUE OF 1.0) CMJT ARG LIST REAL TOP OF GRID - MAXIMUM COARSE MESH J COORDINATE (MAXIMUM VALUE OF 65.0) INUM ARG LIST INTEGER THREE DIGIT NUMBER FOR THE FOLLOWING: HUNDREDS DIGIT = TYPE OF DATA 1 = FIXED POINT 2 = FLOATING POINT 3 = ALPHANUMERIC TENS DIGIT = HEMISPHERIC REFERENCE 0 = NORTHERN HEMISPHERE 1 = SOUTHERN HEMISPHERE UNITS DIGIT = NUMBER OF CHARACTERS TO PLOT MINIMUM = 1 CHARACTER MAXIMUM = 4 CHARACTERS XFAC ARG LIST REAL MAP SCALE FACTOR (DESIRED MAP SCALE = XFAC

  • 1:30,000,000 (STANDARD NMC 65X65 GRID SCALE))

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

TAPE6 PRINTED MAP WHEN EXECUTION IS SUCCESSFUL IERR ARG LIST INTEGER RETURN CODE

SUBPROGRAMS CALLED: NAMES LIBRARY

W3FB04 W3FB05 W3AI15 W3LIB

RESTRICTIONS: BECAUSE THIS CODE COULD PRODUCE CONSIDERABLE OUTPUT THE SUBSET OF THE NMC 65X65 GRID WHICH CAN BE PRINTED IS A FUNCTION OF THE MAP SCALE FACTOR, E.G. FOR XFAC=5 THE MAXIMUM RANGE OF I AND J IS 27.0, FOR XFAC=2 THE RANGE IS 64.0.

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY Y-MP8/832
