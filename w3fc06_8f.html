SUBPROGRAM: W3FC06 WIND DIR AND SPD TO EARTH U,V COMPONENTS AUTHOR: STACKPOLE ORG: W324 DATE: 81-12-30.

ABSTRACT: GIVEN THE WIND DIRECTION AND SPEED, COMPUTE EARTH-ORIENTED (TRUE) WIND COMPONENTS. INPUT DIRECTION AT THE POLE POINT MUST BE CONSISTENT WITH WMO CONVENTIONS, AND OUTPUT COMPONENTS WILL FOLLOW THOSE CONVENTIONS. (SEE OFFICE NOTE 241 FOR WMO DEFINITION.)

PROGRAM HISTORY LOG: 81-12-30 J.STACKPOLE 91-03-06 R.E.JONES CHANGE TO CRAY CFT77 FORTRAN

USAGE: CALL W3FC06 (DIR, SPD, U, V)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

DIR - REAL*4 - WIND DIRECTION, DEGREES SPD - REAL*4 - WIND SPEED, ANY UNITS

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

U - REAL*4 - EARTH-ORIENTED U-COMPONENT V - REAL*4 - EARTH-ORIENTED V-COMPONENT

SUBPROGRAMS CALLED: NAMES LIBRARY

SIN COS SYSTEM

WARNING: THIS CODE WILL NOT VECTORIZE ON CRAY, YOU COULD PUT THE FOUR LINES IN YOUR CODE WITH A COUPLE OF DO LOOPS.

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916-128, Y-MP8/864, Y-MP EL92/256
