SUBPROGRAM: W3FI20 CUT A 65 X 65 GRID TO A NMC 1977 POINT GRID AUTHOR: JONES, R.E.

ORG: W342 DATE: 84-07-02

ABSTRACT: EXTRACTS THE NMC 1977 POINT OCTAGON GRID POINTS OUT OF A 65X65 (4225 POINT) ARRAY.

PROGRAM HISTORY LOG: 73-06-15 R.E.JONES 84-07-02 R.E.JONES CONVERT TO VS FORTRAN 89-02-02 R.E.JONES CONVERT TO MICROSOFT FORTRAN 4.10 90-08-22 R.E.JONES CONVERT TO SUN FORTRAN 1.3 91-03-29 R.E.JONES CONVERT TO SiliconGraphics FORTRAN 93-03-29 R.E.JONES ADD SAVE STATEMENT

USAGE: CALL W3FI20 (A,B)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

A ARG LIST REAL*4 (65 X 65 GRID, 4225 POINT) ARRAY GRID IS OFFICE NOTE 84 TYPE 27 OR 1B HEX

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

B ARG LIST 1977 POINT ARRAY (OCTAGON) OFFICE NOTE 84 TYPE 0 OR HEX 0.

REMARKS: ARRAYS A AND B MAY BE THE SAME ARRAY OR BE EQUIVALENCED, IN WHICH CASE THE FIRST 1977 WORDS OF 'A' ARE WRITTEN OVER.

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916/256, J916/2048.
