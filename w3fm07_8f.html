SUBR: W3FM07 - NINE-POINT SMOOTHER FOR RECTANGULAR GRIDS AUTHOR: CHASE, P.

ORG: W345 DATE: APR 75

ABSTRACT: SMOOTHS DATA ON A RECTANGULAR GRID USING A NINE-POINT SMOOTHING OPERATOR.

PROGRAM HISTORY LOG: 75-04-01 P.CHASE 84-07-01 R.E.JONES CHANGE TO IBM VS FORTRAN 91-04-24 R.E.JONES CHANGE TO CRAY CFT77 FORTRAN

USAGE: CALL W3FM07 (FIN, FOUT, CWORK, GAMMA, NCOL, NROW)

INPUT: ' FIN' - REAL SIZE(NCOL*NROW) ARRAY OF DATA TO BE SMOOTHED ' CWORK' - REAL SIZE(2*NCOL*(NROW+2)) WORK ARRAY ' GAMMA' - COMPLEX SMOOTHING PARAMETER. THE IMAGINARY PART MUST BE POSITIVE. ' NCOL' - INTEGER NUMBER OF COLUMNS IN THE GRID ' NROW' - INTEGER NUMBER OF ROWS IN THE GRID OUTPUT: ' FOUT' - REAL SIZE(NCOL*NROW) ARRAY OF SMOOTHED DATA. MAY BE THE SAME ARRAY AS 'FIN' OR OVERLAP IT IN ANY FASHION.

EXIT STATES: NONE

EXTERNAL REFERENCES: NONE

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY Y-MP8/832
