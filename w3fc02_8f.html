SUBPROGRAM: W3FC02 GRID U,V WIND COMPS.

TO DIR. AND SPEED AUTHOR: STACKPOLE, J. ORG: W324 DATE: 81-12-30

ABSTRACT: GIVEN THE GRID-ORIENTED WIND COMPONENTS ON A NORTHERN HEMISPHERE POLAR STEREOGRAPHIC GRID POINT, COMPUTE THE DIRECTION AND SPEED OF THE WIND AT THAT POINT. INPUT WINDS AT THE NORTH POLE POINT ARE ASSUMED TO HAVE THEIR COMPONENTS FOLLOW THE WMO STANDARDS FOR REPORTING WINDS AT THE NORTH POLE. (SEE OFFICE NOTE 241 FOR WMO DEFINITION). OUTPUT DIRECTION WILL FOLLOW WMO CONVENTION.

PROGRAM HISTORY LOG: 81-12-30 J.STACKPOLE 89-01-20 R.E.JONES CONVERT TO MICROSOFT FORTRAN 4.10 90-06-11 R.E.JONES CONVERT TO SUN FORTRAN 1.3 91-03-30 R.E.JONES CONVERT TO SiliconGRaphics FORTRAN 93-03-29 R.E.JONES ADD SAVE STATEMENT 95-08-09 R.E.JONES COMPILE ON CRAY

USAGE: CALL W3FC02 (FFID, FFJD, FGU, FGV, DIR, SPD)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

FFID ARG LIST REAL*4 I(NORTH POLE) - I(POINT) FFJD ARG LIST REAL*4 J(NORTH POLE) - J(POINT) FGU ARG LIST REAL*4 GRID-ORIENTED U-COMPONENT FGV ARG LIST REAL*4 GRID-ORIENTED V-COMPONENT

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

DIR ARG LIST REAL*4 WIND DIRECTION, DEGREES SPD ARG LIST REAL*4 WIND SPEED

SUBPROGRAMS CALLED: NAMES LIBRARY

ABS ACOS ATAN2 SQRT SYSTEM

WARNING: THIS JOB WILL NOT VECTORIZE ON A CRAY

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916/16256, J916/162048
