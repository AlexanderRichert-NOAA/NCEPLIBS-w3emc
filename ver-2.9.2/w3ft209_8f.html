SUBROUTINE: W3FT209 CONVERT (361,91) GRID TO (101,81) LAMBERT GRID AUTHOR: JONES,R.E.

ORG: W342 DATE: 94-05-18

ABSTRACT: CONVERT A NORTHERN HEMISPHERE 1.0 DEGREE LAT.,LON. 361 BY 91 GRID TO A LAMBERT CONFORMAL 101 BY 81 AWIPS GRIB 209.

PROGRAM HISTORY LOG: 94-05-18 R.E.JONES

USAGE: CALL W3FT209(ALOLA,ALAMB,INTERP)

INPUT ARGUMENTS: ALOLA - 361*91 GRID 1.0 DEG. LAT,LON GRID N. HEMI. 32851 POINT GRID. 360 * 181 ONE DEGREE GRIB GRID 3 WAS FLIPPED, GREENWISH ADDED TO RIGHT SIDE AND CUT TO 361 * 91.
INTERP - 1 LINEAR INTERPOLATION , NE.1 BIQUADRATIC

INPUT FILES: NONE

OUTPUT ARGUMENTS: ALAMB - 101*81 REGIONAL - CENTRAL US MARD DOUBLE RES. (LAMBERT CONFORMAL). 8181 POINT GRID IS AWIPS GRID TYPE 209

OUTPUT FILES: ERROR MESSAGE TO FORTRAN OUTPUT FILE

WARNINGS:

  1. W1 AND W2 ARE USED TO STORE SETS OF CONSTANTS WHICH ARE REUSABLE FOR REPEATED CALLS TO THE SUBROUTINE. 11 OTHER ARRAY ARE SAVED AND REUSED ON THE NEXT CALL.
  2. WIND COMPONENTS ARE NOT ROTATED TO THE 101*81 GRID ORIENTATION AFTER INTERPOLATION. YOU MAY USE W3FC08 TO DO THIS.

RETURN CONDITIONS: NORMAL SUBROUTINE EXIT

SUBPROGRAMS CALLED: UNIQUE : NONE

LIBRARY: W3FB12

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916-128, CRAY Y-MP8/864, CRAY Y-MP EL92/256
