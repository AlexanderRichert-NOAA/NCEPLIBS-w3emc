SUBPROGRAM: W3FA01 COMPUTE LIFTING CONDENDSATION LEVEL AUTHOR: HOWCROFT,J.

ORG: W/NMC342 DATE: 79-07-01

ABSTRACT: GIVEN THE PRESSURE, TEMPERATURE AND RELATIVE HUMIDITY OF AN AIR PARCEL AT SOME POINT IN THE ATMOSPHERE, CALCULATE THE DEWPOINT TEMPERATURE AND THE PRESSURE AND TEMPERATURE OF THE LIFTING CONDENSATION LEVEL.

PROGRAM HISTORY LOG: 79-07-01 J.HOWCROFT 89-01-24 R.E.JONES CHANGE TO MICROSOFT FORTRAN 4.10 90-06-11 R.E.JONES CHANGE TO SUN FORTRAN 1.3 91-03-29 R.E.JONES CONVERT TO SiliconGraphics FORTRAN 93-03-29 R.E.JONES ADD SAVE STATEMENT 95-09-25 R.E.JONES PUT IN CRAY W3 LIBRARY

USAGE: CALL W3FA01(P,T,RH,TD,PLCL,TLCL)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

P ARG LIST PARCEL PRESSURE IN MILLIBARS T ARG LIST PARCEL TEMPERATURE IN DEGREES CELSIUS RH ARG LIST PARCEL RELATIVE HUMIDITY IN PERCENT

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

TD ARG LIST DEWPOINT TEMPERATURE IN DEGREES CELSIUS PLCL ARG LIST PRESSURE OF LCL IN MILLIBARS TLCL ARG LIST TEMPERATURE AT LCL IN DEGREES CELSIUS

SUBPROGRAMS CALLED: NAMES LIBRARY

ABS ALOG ALOG10 EXP SYSLIB

ATTRIBUTES: LANGUAGE: CRAY CFT77 FORTRAN MACHINE: CRAY C916/256, J916/2048
