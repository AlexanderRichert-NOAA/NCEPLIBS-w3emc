SUBPROGRAM: W3AI00 REAL ARRAY TO 16 BIT PACKED FORMAT AUTHOR: JONES,R.E.

ORG: W342 DATE: 85-07-31

ABSTRACT: CONVERTS IEEE FLOATING POINT NUMBERS TO 16 BIT PACKED OFFICE NOTE 84 FORMAT. THE FLOATING POINT NUMBER ARE CONVERTED TO 16 BIT SIGNED SCALED INTEGERS.

PROGRAM HISTORY LOG: 89-10-20 R.E.JONES CONVERT CYBER 205 VERSION OF W3AI00 TO CRAY 90-03-18 R.E.JONES CHANGE TO USE CRAY INTEGER*2 PACKER 90-10-11 R.E.JONES SPECIAL VERSION TO PACK GRIDS LARGER THAN 32743 WORDS. WILL DO OLD AND NEW VERSION. 91-02-16 R.E.JONES CHANGES SO EQUIVALENCE OF PACK AND REAL8 ARRAYS WILL WORK. 93-06-10 R.E.JONES CHANGES FOR ARRAY SIZE (512,512) 262144 WORDS. 98-03-10 B. VUONG REMOVE THE CDIR$ INTEGER=64 DIRECTIVE 98-11-18 Gilbert Changed to pack IEEE values for the IBM SP

USAGE: CALL W3AI00 (REAL8, PACK, LABEL)

INPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

REAL8 ARG LIST ARRAY OF CRAY FLOATING POINT NUMBERS LABEL ARG LIST SIX 8-BYTE INTEGER WORDS. MUST HAVE FIRST 8 OF 12 32 BIT WORD OFFICE NOTE 84 LABEL. WORD 6 MUST HAVE IN BITS 31-00 THE NUMBER OF REAL WORDS IN ARRAY REAL8 IF J IS GREATER THAN 32743. J IN BITS 15-0 OF THE 4TH ID WORD IS SET ZERO.

OUTPUT VARIABLES: NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES

PACK ARG LIST PACKED OUTPUT ARRAY OF INTEGER WORDS OF SIZE 6 + (J+3)/4 , J = NO. POINTS IN LABEL (FROM WORD 4 BITS 15-00). LABEL WILL BE COPIED TO PACK WORDS 1-4. PACK WILL CONTAIN THE FOLLOWING IN WORDS 5-6 WORD 5 BITS 63-48 NUMBER OF BYTES IN WHOLE RECORD. WILL NOT BE CORRECT IF J > 32743. WORD 5 BITS 47-32 EXCLUSIVE-OR CHECKSUM BY 16 BIT WORDS OF WHOLE ARRAY PACK EXCLUDING CHECKSUM ITSELF. WORD 5 BITS 31-00 CENTER VALUE A = MEAN OF MAX AND MIN VALUES. CONVERTED TO IBM 32 FLOATING POINT NUMBER. WORD 6 BITS 63-48 ZERO. WORD 6 BITS 47-32 16 BIT SHIFT VALUE N. THE LEAST INTEGER SUCH THAT ABS(X-A)/2**N LT 1 FOR ALL X IN REAL8. LIMITED TO +-127. WORD 6 BITS 31-00 NUMBER OF WORDS IN REAL8 IF > 32743, RIGHT ADJUSTED IF <= 32743 SET ZERO.

SUBPROGRAMS CALLED: NAMES LIBRARY

IAND IOR BTEST SYSTEM

REMARKS: PACK AND LABEL MAY BE EQUIVALENCED. N, THE NUMBER OF POINTS IN A GRID IS NOW IN 32 BIT ID WORD 12.

ATTRIBUTES: LANGUAGE: IBM XL FORTRAN. MACHINE: IBM SP
